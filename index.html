Javascript Quick Start

View is a lightweight and easy-to-use framework.



Make it as simple as possible, but not simpler.

Albert Einstein

Getting up and running is a quick process. First, you have to setup your workflow. If we were implementing View into a Javascript canvas demo start like this: 

<!DOCTYPE html>
<html>
  <head>
    <canvas id="canvas" width="600" height="600"></canvas>
    <script src="build/view.min.js"></script>
  </head>
  <body>
    <script>
    // code goes here:
    </script>
  </body>
</html>

Now we have loaded view.js and created the canvas, but when you run this code you'll see a blank page. We need to setup the Javascript side to make our program do anything.

var canvas = document.getElementById("canvas"); /* Get canvas element from dom. */
var ctx = canvas.getContext("2d"); /* We will use this to add things to the canvas. */
var app = new VIEW.ViewPort(0,0); /* Setup the View Camera */

var viewSquare = app.addObj(300, 300, null); /* add the square to the view object tracker */
var squareID = viewSquare[0]
var square = ctx.fillRect(app.getX(squareID), app.getY(squareID), 50, 50); /* create a square */

function animate() { /* animation loop */
  ctx.clearRect(0,0,600,600); /* clear canvas */
  app.updateStats(4); /* Update Viewjs status */
  square = ctx.fillRect(app.getX(squareID), app.getY(squareID), 50, 50);
  window.requestAnimationFrame(animate); /* run animation loop */
}
animate(); /* ANIMATE! */

You should see a black square

Now, lets see whether this works. Open the html file you created in a browser (preferably Google Chrome). If you are using Google Chrome, click Ctrl + Shift + J to go into the developer console. In Mozilla Firefox and Microsoft Edge hold Fn + F11. If the doesn't work, try holding F11. In Chrome/Firefox you may see a rainbow banner stating which version of View you are using, don't pay attention to that. Type the function to move to the left: app.setDirection("left"); You will see the cube move out of sight. Actually, it wasn't the cube that moved away.. YOU moved away. You just made a simple View program! Try the live demo here.

Python Quick Start

Note: We assume that you already have Pygame installed on your computer. Pygame is a python library to create graphical games and interfaces.

View is nearly identical between Python and Javascript. So you won't need to worry about different function parameters and things like that. Now, type the following code to setup Pygame.

import pygame, sys # Load Pygame
import view
pygame.init() # Initializes Pygame
app = view.ViewPort(0, 0) # Don't worry about the arguments in ViewPort just yet
screen = pygame.display.set_mode((600, 600)) # Create Pygame window
viewSquare = app.ViewPort(300, 300, None) # Adds square to View
squareID = viewSquare[0]
squareData = pygame.Rect(app.getX(squareID), app.getY(squareID), 50, 50)
while True:
  for event in pygame.event.get():
    if event.type == pygame.QUIT:
      sys.exit()
    elif event.type == pygame.KEYDOWN:
      if event.key == pygame.K_SPACE:
        app.setDirection("left")
    screen.fill((255, 255, 255)) # Clears the screen
    app.updateStats(4)
    squareData = pygame.Rect(app.getX(squareID), app.getY(squareID), 50, 50)
    pygame.draw.rect(screen, [0, 0, 0], squareData, 0)
    pygame.display.flip()
pygame.quit()
